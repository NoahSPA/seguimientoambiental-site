# Imágenes Necesarias para la Landing Page

Este documento lista todas las imágenes que puedes agregar a la carpeta `public/images/` para mejorar la apariencia visual del sitio.

## Imágenes Requeridas

### Hero Section
- **hero-image.jpg** (recomendado: 1200x800px)
  - Imagen principal de la sección hero
  - Debe representar seguimiento ambiental, monitoreo, o naturaleza
  - Formato: JPG, optimizado para web

### Servicios (Backgrounds opcionales)
- **service-1.jpg** hasta **service-6.jpg** (recomendado: 800x600px)
  - Imágenes de fondo para las tarjetas de servicios
  - Temas sugeridos: estudios ambientales, monitoreo, auditorías, etc.

### Galería de Servicios
- **gallery-1.jpg** (recomendado: 800x600px)
  - Tema: Monitoreo ambiental
  
- **gallery-2.jpg** (recomendado: 800x600px)
  - Tema: Consultoría ambiental
  
- **gallery-3.jpg** (recomendado: 800x600px)
  - Tema: Cumplimiento normativo

### Sección CTA
- **cta-background.jpg** (recomendado: 1920x1080px)
  - Imagen de fondo para la sección de contacto
  - Opcional, se usa con baja opacidad

## Notas

- Todas las imágenes tienen fallbacks automáticos si no existen
- Las imágenes se optimizan automáticamente
- Formatos recomendados: JPG para fotos, PNG para gráficos
- Tamaños sugeridos son orientativos, el sitio es responsive

## Ubicación

Coloca todas las imágenes en: `public/images/`

Ejemplo de estructura:
```
public/
  images/
    hero-image.jpg
    service-1.jpg
    service-2.jpg
    service-3.jpg
    service-4.jpg
    service-5.jpg
    service-6.jpg
    gallery-1.jpg
    gallery-2.jpg
    gallery-3.jpg
    cta-background.jpg
```
